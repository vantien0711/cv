ChatGPT said:

Để chỉnh sửa một section cụ thể trong class main mà không ảnh hưởng đến các section khác, bạn có thể sử dụng một trong các cách sau:

1. Dùng class riêng biệt cho từng section

Nếu chưa có, hãy thêm một class riêng cho section mà bạn muốn chỉnh sửa:

html
<section class="portfolio-section photo-agency">
    ...
</section>
<section class="portfolio-section restaurant-design">
    ...
</section>
<section class="portfolio-section ecommerce">
    ...
</section>

Sau đó, trong CSS, bạn có thể chọn riêng section cần chỉnh sửa:

css
.photo-agency {
    background-color: #f0f0f0; /* Chỉ ảnh hưởng đến phần Photo Agency */
    padding: 20px;
}

2. Dùng nth-child() nếu không thể thêm class

Nếu bạn không thể thêm class mới vào HTML, bạn có thể sử dụng nth-child() dựa trên vị trí của section trong main:

css
.main section:nth-child(1) {
    background-color: #f0f0f0;
}

Trong ví dụ trên, nth-child(1) chọn section đầu tiên (Photo Agency Website).

3. Dùng id nếu chỉ có một phần tử duy nhất cần chỉnh

Nếu chỉ cần chỉnh sửa một section cụ thể và nó có một id, bạn có thể làm thế này:

html
<section id="photo-agency">
    ...
</section>

Rồi trong CSS:

css
#photo-agency {
    border: 2px solid red;
}

Cách nào phù hợp?

  • Cách 1 (thêm class riêng): Tốt nhất nếu bạn có quyền chỉnh sửa HTML.

  • Cách 2 (nth-child()): Dùng khi không thể thêm class nhưng cần chọn theo vị trí.

  • Cách 3 (id): Dùng khi chỉ có một section duy nhất cần chỉnh.

Để đưa chữ nằm ngang bên cạnh hình ảnh, bạn có thể sử dụng CSS Flexbox hoặc CSS Grid. Dưới đây là cách thực hiện bằng Flexbox:

1. HTML (Cấu trúc lại)

html
<section class="hobby-section">
    <div class="hobby-container">
        <img src="your-image.jpg" alt="Sở thích" class="hobby-image">
        <h2 class="hobby-text">Sở Thích</h2>
    </div>
</section>

2. CSS (Sắp xếp ngang hàng)

css
.hobby-section {
    background-color: #e0e0e0; /* Màu nền */
    padding: 20px;
    text-align: center;
}

.hobby-container {
    display: flex;
    align-items: center; /* Căn giữa theo chiều dọc */
    justify-content: center; /* Căn giữa theo chiều ngang */
    gap: 20px; /* Khoảng cách giữa chữ và ảnh */
}

.hobby-image {
    width: 100px; /* Kích thước ảnh */
    height: 100px;
    border-radius: 50%;
    object-fit: cover;
}

.hobby-text {
    font-size: 24px;
    font-weight: bold;
}

Giải thích:

  • display: flex; giúp chữ và ảnh nằm ngang trên cùng một hàng.

  • align-items: center; giúp căn giữa theo chiều dọc.

  • gap: 20px; tạo khoảng cách giữa ảnh và chữ.

  • justify-content: center; giúp căn giữa theo chiều ngang.
